# Kai Workspace Guide

## Getting Started

#### 1. Install Xcode developer tools

```bash
xcode-select --install

# Optional: Set HostName
sudo scutil --set HostName MOD
```

#### 2. Clone Config

```bash
git clone https://github.com/connect-kai/config ~/Config
```

#### 3. Set up Homebrew

1. Install Homebrew: https://brew.sh
2. Install important apps and fonts [Homebrew](https://github.com/connect-kai/config/tree/master/Homebrew) section

#### 4. Set up [System Defaults](https://github.com/connect-kai/config/tree/master/System)

#### 5. Set up [Zsh](https://github.com/connect-kai/config/tree/master/Zsh)

#### 6. Set up [Git](https://github.com/connect-kai/config/tree/master/Git)

#### 7. Set up [Vim](https://github.com/connect-kai/config/tree/master/Nvim)

#### 8. Other apps

Now go through other sections relevant to you, and set them up one by one.
